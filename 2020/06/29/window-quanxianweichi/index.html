CMD 通过 sc 进行注册

命令详解：sc create 服务名 binpath= “命令” start= “auto” obj= “objectname”

PowerShell 通过New-service 创建
注册表键值修改
HKCU

计算机\HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Run

HKLM

计算机\HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Run

Userinit 注册键

计算机\HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows NT\CurrentVersion\Winlogon

powershell

LogonScript 注册键

ogonScript 的启动优先级较高，甚至优于某些杀软启动

Explore\run 注册键(需要高权限)

HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Policies\Explorer 与 HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Policies\Explorer

cmd

powershell

Command Processor\AutoRun 注册键

HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Command Processor

启动时间先于用户登录

Winlogon\taskman 注册键

HKLM\SOFTWARE\Microsoft\Windows NT\CurrentVersion\Winlogon 与 HKCU\SOFTWARE\Microsoft\Windows NT\CurrentVersion\Winlogon，但只有 HKLM 注册表下的 taskman 是生效的

Windows NT\CurrentVersion\Windows\run 注册键

HKCU:\Software\Microsoft\Windows NT\CurrentVersion\Windows

Windows NT\CurrentVersion\Windows\load 注册键

HKCU:\Software\Microsoft\Windows NT\CurrentVersion\Windows 下和 Windows\run 一样，只有 HKCU 下的该键才生效
